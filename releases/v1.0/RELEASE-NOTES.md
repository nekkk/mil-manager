# v1.0

Lancamento oficial do MIL Manager.

## Entregas principais

- catalogo sanitizado separado do codigo-fonte
- delivery publico sem `downloadUrl` real nos JSONs publicos
- suporte a traducoes, modificacoes, trapacas e jogos salvos
- instalacao individual de trapacas e saves
- sincronizacao de biblioteca para emuladores
- internacionalizacao por arquivos em `romfs/lang`

## Arquivos desta release

- `mil_manager.nro`
- `mil_manager_forwarder.nsp`
